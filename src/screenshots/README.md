## How to Take Screenshots:

### Method 1: Browser Screenshot (Recommended)
1. Open each website in your browser
2. Press F12 to open Developer Tools
3. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
4. Type "screenshot" and select "Capture full size screenshot"
5. Save the image with the corresponding name

### Method 2: Online Tool
1. Use a website screenshot tool like:
   - https://www.screenshotmachine.com/
   - https://www.websiteplanet.com/webtools/screenshot-generator/
2. Enter the URL
3. Download and save with the correct filename

### Method 3: Manual Screenshot
1. Open the website
2. Press Print Screen or use Snipping Tool (Windows) / Cmd+Shift+4 (Mac)
3. Crop to show just the website content
4. Save as JPG with the correct filename

## Image Requirements:
- Format: JPG or JPEG
- Recommended size: 1200px width minimum
- Aspect ratio: 16:9 or similar (landscape)
- File size: Keep under 500KB for faster loading

Once you add these images, your portfolio will automatically display them instead of the gradient backgrounds!
